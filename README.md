# django-socialmedia
Used Tech Stack

1.	Django
2.	Sqlite
3.	Web Socket (Django Channel)
4.	Html

Current Features:

1.	sign up/sign in
2.	Create post and caption
3.	Like posts
4.	Follow other users
5.	Feed of only followed users
